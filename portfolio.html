No Fear Beer
September, 2018

Wanna grab a beer in Philly tonight? Make sure you use No Fear Beer first! No Fear Beer is an application that uses Google Places to find beer in your favorite Philly neighborhood. Once you pick a place, we use OpenDataPhilly to show you the most recent crimes nearby - we always want you to keep safety in mind when you're planning a big night out.

This project was created with Daland Dixon, Anders Engman, and Tim Fisher

Working on this project was a pleasure and a huge learning experience. It was my first time working with a team on a coding project so there were some growing pains (mostly GitHub pains at first). I got to work with three very talented people who I had never met before and we were able to reach our end goal of building a fun application for class.

Technologies Used: HTML, CSS, Bootstrap, JavaScript

LIRI
October, 2018

LIRI is like iPhone's SIRI! LIRI is a Language Interpretation and Recognition Interface. LIRI is a command line node app that takes in parameters and gives you back data.

LIRI pulls data from the OMDB, Bands in Town, and Spotify APIs based on user input.

Technologies Used: JavaScript, MySQL, Node.js

Train Time Schedule
September, 2018

Choo Choo! This is a totally imaginary train schedule! This is an application that calculates train times based on stored information from previous users.

Every time a user adds a train to the schedule its information is stored to a Firebase database. Using the user data and Moment.js, the application returns the current train schedule.

Technologies Used: HTML, CSS, Bootstrap, JavaScript, Moment.js, Firebase
