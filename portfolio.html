Bamazon has three different programs: Bamazon Customer, Bamazon Manager, and Bamazon Supervisor. Each of these programs allows the user to interact with the Bamazon database in different ways.

LIRI
October, 2018

Hey Liri? LIRI is just like SIRI; however, while SIRI is a Speech Interpretation and Recognition Interface, LIRI is a Language Interpretation and Recognition Interface. LIRI is a command line node app that takes in parameters and gives you back data.

NO FEAR BEER
September, 2018

Wanna grab a beer in Philly tonight? Make sure you use No Fear Beer first! No Fear Beer is an application that uses Google Places to find beer in your favorite Philly neighborhood. Once you pick a place, we use OpenDataPhilly to show you the most recent crimes nearby - we always want you to keep safety in mind when you're playing your big night out.

This project was created by Daland Dixon, Anders Engman, Tim Fisher, and Brooke Mitchell - that's me!

Working on this project was a pleasure and a huge learning experience. It was my first time working with a team on a coding project so there were some growing pains (mostly GitHub pains at first), but overall it went extremely well. I got to work with three very talented people who I had never met before and we were able to reach our end goal of building a fun application for class.

I was the head of front end design for this project. I wrote the HTML for the application and used Bootstrap along with custom CSS to style the page. I also worked with Javascript and jQuery to create the sign out function.

Train Time Schedule
September, 2018

Choo Choo! This is a totally imaginary train schedule! This is an application that calculates train times based on stored information from previous users.

Every time a user adds a train to the schedule its information is stored to a Firebase database. Using the user data and Moment.js, the application returns the current train schedule.

Check it out and add your own train to the schedule!

NOTE: others will see what you submit, please keep it clean and SFW, or at least not offensive
